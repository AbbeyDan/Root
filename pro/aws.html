了解 AWS 云产品

若你对AWS了解不多，这将是你学习AWS服务操作指南与AWS服务应用课程之前的必备课程。我们会为你深入解读AWS云产品。

Amazon Web Services ( AWS )

11年来，AWS一直是世界上服务和应用广泛的云服务平台。通过位于全球16个不同区域的43个可用区（AZ），AWS向客户提供90多项功能全面的服务并获得了全球数百万客户的青睐。通过AWS的服务，客户能够有效地强化自身基础设施，提高敏捷度，降低成本。
